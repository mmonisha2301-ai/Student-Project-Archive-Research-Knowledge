# 🎓 Student Project Repository System (SPARK)

## 📌 Description
A web-based application developed using PHP and MySQL to manage and explore student academic projects with secure login and role-based access.

## 🚀 Features
- User Registration with College Email Verification
- Role-based Login (Student / Staff)
- Upload & Manage Projects (Staff Only)
- Batch-wise Project Filtering
- Secure Authentication with Password Hashing
- File Upload & Viewing System

## 🛠️ Technologies Used
- PHP
- MySQL
- HTML, CSS
- XAMPP

## 🔐 Security Features
- Password Hashing
- Session Management
- Role-based Access Control
